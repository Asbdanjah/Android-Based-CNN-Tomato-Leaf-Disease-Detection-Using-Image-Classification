# ML Model Documentation

## Model Architecture
- MobileNetV2 (transfer learning)
- 98.2% validation accuracy
- 10 disease classes

## Training
- Dataset: PlantVillage (10,000 images)
- Epochs: 10
- Optimizer: Adam (lr=0.001)

## Performance
- Accuracy: 98.2%
- Inference: 50-100 ms
- Model Size: 3 MB (quantized)
